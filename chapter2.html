Chapter 2: Probability

Definition of Probability

Probability is the likelihood that an event will occur (0 ≤ P ≤ 1).

Types of Events

  • Simple Event
  • Compound Event
  • Independent & Dependent Events
  • Mutually Exclusive Events

Addition & Multiplication Rules

Used to calculate probability of combined events.

Conditional Probability

Probability of an event given another event has occurred: P(A|B) = P(A∩B)/P(B)

Bayes’ Theorem

Updates probability based on prior knowledge: P(A|B) = [P(B|A) × P(A)] / P(B)
