Chapter 2: Probability

Definition of Probability

Probability is the likelihood that an event will occur (0 ≤ P ≤ 1). It is defined as the ratio of the number of favorable outcomes to the total number of equally likely outcomes. 

Mathematical Definition:
P(A) = Number of favorable outcomes / Total number of outcomes

P(A) = 0 → Event is impossible
P(A) = 1 → Event is certain
0 < P(A) < 1 → Event is uncertain

Types of Events

  • Simple Event
  • Compound Event
  • Independent & Dependent Events
  • Mutually Exclusive Events

Addition & Multiplication Rules

Probability can be interpreted in different ways depending on the experiment and available information:

  1. Classical Interpretation: Applies when all outcomes are equally likely. 
    Formula: P(A) = n(A) / n(S)

Example:
Experiment: Tossing a fair die
Event A: Getting an even number
Favorable outcomes = {2, 4, 6}
Total outcomes = 6
P(A) = 3/6 = 1/2
  2. Empirical (Relative Frequency) Interpretation: Probability is the ratio of the number of times an event occurs to the total trials in repeated experiments. 
    Formula: P(A) = lim (Number of times A occurs / Total trials)

Example:
A machine produces defective items 20 times in 1000 trials
Probability of defect = 20 / 1000 = 0.02
  3. Subjective Interpretation: Based on personal judgment, experience, or belief. 
    Example:
Probability of rain tomorrow based on weather forecast = 0.7
Probability of project completion on time based on experience = 0.8

Applications of Probability

  • Risk analysis and reliability engineering
  • Machine learning and artificial intelligence
  • Network traffic modeling
  • Quality control and defect prediction
  • Weather forecasting and decision-making
  • Financial modeling and investment analysis

Probability enables informed decisions under uncertainty and optimizes system performance. Conditional probability formula: P(A|B) = P(A ∩ B) / P(B)

Bayes’ Theorem

Bayes’ Theorem provides a method for updating the probability of an event based on new information. It is widely used in Bayesian statistics, machine learning, AI, and decision-making systems. 

Statement:
If A and B are two events such that P(B) ≠ 0, then

P(A | B) = [ P(B | A) × P(A) ] / P(B)

This formula allows computation of the probability of a cause (A) given an observed effect (B).
