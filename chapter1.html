Chapter 1: Description of Data

Introduction

Data refers to raw facts and observations collected from experiments, surveys, or real-life situations. The main objective of statistics is to organize, analyze, and interpret data in order to draw meaningful conclusions and support decision-making.

Types of Variables

A variable is a characteristic, property, or quantity that can take different values for different observations or over time. Variables are used in statistics to represent data that can change and be measured or classified during data collection and analysis. They are classified into the following types:

  • Qualitative (Categorical): Non-numerical data such as gender, color, or category.
  • Quantitative: Numerical data which can be:
    • Discrete: Countable values (e.g., number of students).
    • Continuous: Measured values (e.g., height, weight).

Mean and Weighted Mean

The mean The mean is a measure of central tendency obtained by adding all the observations in a data set and dividing the sum by the total number of observations. It helps in identifying the spread, central value, and presence of outliers in the data. Formula: Mean = (Sum of all observations) / (Number of observations)

The weighted meanThe weighted mean is a measure of central tendency in which each observation is multiplied by a corresponding weight that reflects its importance. The sum of these weighted values is divided by the sum of the weights. Formula: Weighted Mean = (Σ wᵢxᵢ) / (Σ wᵢ) where xᵢ are the observations and wᵢ are their respective weights.

Median and Mode

The median is the middle value when data is arranged in ascending or descending order.The median is a measure of central tendency that represents the middle value of a data set when the observations are arranged in ascending or descending order. If the number of observations is even, the median is the average of the two middle values.Median for Discrete Data The median is the middle value when data is arranged in ascending or descending order. Steps: Arrange data in order If n is odd → Median = middle value If n is even → Median = average of two middle values Example: Data: 15, 20, 25, 30, 35 Median = 25 Median for Continuous (Grouped) Data The median is calculated using the median class and cumulative frequency. Formula: Median = l + [( (N/2) − cf ) / f ] × h Where: l = lower boundary of median class N = total frequency cf = cumulative frequency before median class f = frequency of median class h = class width Median is preferred for skewed data such as income, network latency, or response time analysis.

The mode is the value that occurs most frequently in the data set.The mode is the value that occurs most frequently in a data set. A data set may have one mode (unimodal), more than one mode (bimodal or multimodal), or no mode at all.Mode for Discrete Data Mode is the value that occurs most frequently in a dataset. Example: Data: 2, 4, 4, 6, 8 Mode = 4 Mode for Continuous (Grouped) Data Mode is calculated using the modal class (class with highest frequency). Formula: Mode = l + [(f₁ − f₀) / (2f₁ − f₀ − f₂)] × h Where: l = lower boundary of modal class f₁ = frequency of modal class f₀ = frequency of preceding class f₂ = frequency of succeeding class h = class width Mode is especially useful for categorical data and market demand analysis.

Quartiles

Quartiles divide the data into four equal parts: Q1 (lower quartile), Q2 (median), and Q3 (upper quartile). They help in understanding the spread and distribution of data.Quartiles are measures of position that divide an ordered data set into four equal parts. Each part represents one-fourth (25%) of the total data. First Quartile (Q1): The value below which 25% of the observations lie. Second Quartile (Q2): The median of the data; 50% of the observations lie below this value. Third Quartile (Q3): The value below which 75% of the observations lie. Quartiles help in understanding the spread and distribution of data.

Variance

Variance measures the degree of dispersion of data points from the mean. A higher variance indicates that data points are more spread out.Variance is a measure of dispersion that indicates how far the observations in a data set are spread out from the mean. It is calculated as the average of the squared differences between each observation and the mean.A higher variance shows that the data values are more spread out, while a lower variance indicates that the data values are closer to the mean.

Coefficient of Variation

The Coefficient of Variation is a relative measure of dispersion that expresses the standard deviation as a percentage of the mean. It is used to compare the degree of variation between different data sets, even when their means are different. Formula: Coefficient of Variation (CV) = (Standard Deviation / Mean) × 100 A higher CV indicates greater variability in the data, while a lower CV indicates more consistency. The coefficient of variation (CV) is a relative measure of dispersion. It is expressed as:
CV = (Standard Deviation / Mean) × 100

Skewness

Skewness describes the symmetry of a data distribution. A distribution can be positively skewed, negatively skewed, or symmetric.Skewness is a measure of the asymmetry of the distribution of data around its mean. It indicates whether the data distribution is symmetric or skewed to the left or right. Positive Skewness: The right tail is longer; mean > median > mode. Negative Skewness: The left tail is longer; mean < median < mode. Zero Skewness: The distribution is symmetric.

A histogram is a graphical representation of frequency distribution using bars. It helps visualize the shape and spread of data.A histogram is a graphical representation of the frequency distribution of a continuous data set. It uses adjacent rectangles (bars) to show the number of observations falling within specified class intervals. The width of each bar represents the class interval, and the height represents the frequency.A histogram represents the frequency distribution of continuous data. Data is grouped into class intervals, and rectangles are drawn whose heights represent frequencies. Box Plot

A box plot summarizes data using minimum value, quartiles, median, and maximum value. It is useful for detecting outliers. A box plot is a graphical method used to display the distribution of data based on the five-number summary: minimum, first quartile (Q1), median, third quartile (Q3), and maximum.
