Chapter 1: Description of Data

Introduction

Data refers to raw facts and observations collected from experiments, surveys, or real-life situations. The main objective of statistics is to organize, analyze, and interpret data in order to draw meaningful conclusions and support decision-making.

Types of Variables

A variable is a characteristic, property, or quantity that can take different values for different observations or over time. Variables are classified into:

  • Qualitative (Categorical): Non-numerical data such as gender, color, or category.
  • Quantitative: Numerical data, which can be:
    • Discrete: Countable values (e.g., number of students).
    • Continuous: Measured values (e.g., height, weight).

Mean and Weighted Mean

Mean: The average value of the data, obtained by summing all observations and dividing by the number of observations. 

Mean = (Sum of all observations) / (Number of observations)

Weighted Mean: Each observation is multiplied by a corresponding weight reflecting its importance. The sum of weighted values is divided by the sum of the weights. 

Weighted Mean = (Σ wᵢxᵢ) / (Σ wᵢ)
where xᵢ = observations, wᵢ = weights

Median and Mode

Median: Middle value when data is arranged in order. If even number of observations, median is the average of two middle values. 

Steps:
1. Arrange data in order
2. If n is odd → Median = middle value
3. If n is even → Median = average of two middle values
Example:
Data: 15, 20, 25, 30, 35
Median = 25

For continuous data: 

Median = l + [( (N/2) − cf ) / f ] × h
Where:
l = lower boundary of median class
N = total frequency
cf = cumulative frequency before median class
f = frequency of median class
h = class width

Mode: The value occurring most frequently. Can be unimodal, bimodal, or multimodal. 

Example (Discrete Data):
Data: 2, 4, 4, 6, 8
Mode = 4

Mode (Continuous Data):
Mode = l + [(f₁ − f₀) / (2f₁ − f₀ − f₂)] × h
Where:
l = lower boundary of modal class
f₁ = frequency of modal class
f₀ = frequency of preceding class
f₂ = frequency of succeeding class
h = class width

Quartiles

Quartiles divide data into four equal parts: Q1 (25%), Q2 (50%, median), Q3 (75%). They help understand the spread and distribution of data.

Variance

Variance measures how spread out the data points are from the mean. Higher variance = more spread; lower variance = data closer to mean.

Coefficient of Variation (CV)

CV is a relative measure of dispersion: CV = (Standard Deviation / Mean) × 100. It helps compare variability between datasets with different means.

Skewness

Skewness describes the symmetry of data distribution:

  • Positive Skewness: Right tail longer; mean > median > mode.
  • Negative Skewness: Left tail longer; mean < median < mode.
  • Zero Skewness: Symmetric distribution.

Histogram

A histogram is a bar graph representing frequency distribution of continuous data. Class intervals are on x-axis; frequency is on y-axis. 

Example:
Marks: 40, 45, 50, 55, 60, 60, 65, 70, 75, 80
Intervals (width 10):
40–50: 2
50–60: 3
60–70: 3
70–80: 2

Box Plot

A box plot (box-and-whisker) summarizes data using five-number summary: minimum, Q1, median (Q2), Q3, maximum. 

Example:
Data: 10, 20, 25, 30, 35, 40, 45
Min = 10, Q1 = 20, Median = 30, Q3 = 40, Max = 45
