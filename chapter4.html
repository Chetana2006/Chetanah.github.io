Chapter 4: Statistical Inference

Sampling Methods

Select representative data from a population using methods such as Random, Stratified, or Systematic sampling. Probability sampling ensures every unit has a known, non-zero chance of being selected, producing unbiased and reliable results.

Types of Probability Sampling:

  • Simple Random Sampling
  • Systematic Sampling
  • Stratified Sampling
  • Cluster Sampling

Confidence Intervals

A confidence interval estimates an unknown population parameter (mean, proportion) within a specified confidence level (commonly 90%, 95%, 99%). 

General Formula:
Confidence Interval = Point Estimate ± Margin of Error
Margin of Error = Critical Value × Standard Error

Sample Size Determination

Sample size determination ensures reliable statistical results:

  • Too small → unreliable results
  • Too large → increases cost and time
  • Correct sample size → balances accuracy and efficiency

Sample Size for Estimating Mean:

Formula:
n = ( zα/2 × σ / E )²

Where:
n = required sample size
zα/2 = critical value for confidence level
σ = population standard deviation
E = allowable error (margin of error)

Example:

Given:
σ = 10, E = 2, Confidence level = 95%
zα/2 = 1.96

n = (1.96 × 10 / 2)²
n = (9.8)² = 96.04
Required Sample Size: n ≈ 97 (rounded up)

Hypothesis Testing

Hypothesis testing is a decision-making process using sample data to accept or reject a hypothesis about a population parameter.

Z-test & t-test

These tests compare sample and population means:

  • Z-Test: Used when sample size is large (n ≥ 30) or population standard deviation (σ) is known. Data should be approximately normally distributed and observations independent.
  • t-Test: Used when sample size is small (n < 30) and population standard deviation is unknown. Assumes data is approximately normally distributed.
